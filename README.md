# 🎥 YouTube RAG Chatbot

Ask questions about any YouTube video using Retrieval-Augmented Generation (RAG).

## 🚀 Features
- Extracts YouTube transcripts
- Uses RAG to answer questions
- Grounded answers with timestamps
- Clickable video segments
- Streamlit UI

## 🛠 Tech Stack
- LangChain
- Google Gemini (LLM)
- HuggingFace Embeddings
- FAISS (Vector DB)
- Streamlit
